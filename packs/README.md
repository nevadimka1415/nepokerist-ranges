# Паки спектров

Пак — это набор спектров, который **уже лежит в приложении** у любого, кто его открыл.
Файл `nepokerist.json` тянется приложением напрямую из этого репозитория, поэтому
**чтобы выложить новые спектры всем, достаточно закоммитить сюда файл — релиз приложения не нужен.**

## Как заменить примеры своими спектрами

1. Собери спектры в приложении, в папке «Мои спектры».
2. Нажми **Экспорт JSON** — скачается файл проекта.
3. Открой его и достань нужное (см. формат ниже) либо попроси меня собрать пак из твоего экспорта.
4. Подними `version` на единицу.
5. Закоммить `packs/nepokerist.json` в `main`.

Через несколько минут новые спектры появятся у всех: приложение проверяет пак при каждом запуске.

## Формат

```jsonc
{
  "id": "nepokerist-core",        // не меняй: по нему находится папка пака
  "name": "Спектры Непокериста",  // название папки в библиотеке
  "version": 1,                   // ← поднимай при каждом обновлении, иначе не подтянется
  "updatedAt": "2026-07-15",
  "actions": [                    // действия, на которые ссылаются руки
    { "id": "pack-nepokerist-raise", "color": "#ef476f", "label": "Рейз" }
  ],
  "folders": [
    {
      "id": "pack-nepokerist-examples",
      "name": "Примеры (замени своими)",
      "color": "#8ecae6",
      "folders": [],
      "items": [
        {
          "id": "pack-range-pairs",   // ← уникальный и НЕИЗМЕННЫЙ идентификатор спектра
          "name": "Все пары",
          "hands": { "AA": "pack-nepokerist-raise", "KK": "pack-nepokerist-raise" },
          "createdAt": 1768435200000,
          "updatedAt": 1768435200000
        }
      ]
    }
  ]
}
```

Подпись руки: пара — `AA`, одномастная — `AKs`, разномастная — `AKo`.
Два действия в одной руке (сплит) — `id1||id2`.

## Правила, по которым работает подсев

Это важно понимать, чтобы не сломать людям библиотеку:

- **Только добавление.** Спектр подсевается один раз — по своему `id`. Уже подсеянные id
  запоминаются в `poker_ranges_seeded_range_ids_v1`.
- **Правки пользователя неприкосновенны.** Если человек изменил подсеянный спектр,
  обновление пака его не перезапишет.
- **Удаление уважается.** Удалил спектр — он не воскреснет.
- **Новый спектр = новый `id`.** Если поменять `id` существующего спектра, он подсеется
  повторно как новый и появится дубликатом. Не меняй `id` без нужды.
- **Действия сопоставляются по названию.** У каждого пользователя свои внутренние id для
  «Рейз»/«Колл», поэтому пак ищет действие с таким же названием и переиспользует его.
  Если такого нет — добавляет своё. Поэтому называй действия привычно.
